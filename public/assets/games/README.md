# Game Assets Directory

## Memes Directory (`/memes`)
Place your meme images here. Each image should be named according to the answer.

**Naming Convention:**
- Use lowercase, replace spaces with underscores
- Example: `distracted_boyfriend.jpg`, `drake_hotline_bling.png`

## Pornstars Directory (`/pornstars`)
Place performer images here. Each image should be named with the performer's name.

**Naming Convention:**
- Use lowercase, replace spaces with underscores  
- Example: `riley_reid.jpg`, `mia_khalifa.png`

**Supported Formats:** `.jpg`, `.jpeg`, `.png`, `.webp`
**Resolution:** Minimum 800x600px | File Size: Under 500KB
